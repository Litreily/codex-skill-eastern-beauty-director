- 国家地理专题

## Output Rule

When `Type=Cover` or `页面类型=封面`:

- Output Cover Prompt.
- Use Cover Template V1.0.
- Apply V1.1 Cover Coordinate Lock.
- Keep 3:4 ratio and 1080x1440.
- Include series identity, issue number, theme title, subtitle, one character or representative visual, and one main element.

When `Type=Overview` or `页面类型=总览页`:

- Output Overview Prompt.
- Use Overview Template V3.0.
- Apply V1.1 Overview Coordinate Lock.
- Use 2x2 character grid.
- Keep uniform scale and visual weight.
- Do not repeat the cover column frame, logo, or issue block.

When `Type=Character` or `页面类型=人物页`:

- Output Character Prompt.
- Use Character Page Template V4.0.
- Must apply Character Page Layout System V4.0 Locked.
- Use the character parameters only; do not redesign the template.
- Keep the information zone as overlay, not a white board or independent panel.
- Keep character and text in one continuous visual space.

All atlas prompts must follow V1.1 Prompt Generation Order.
