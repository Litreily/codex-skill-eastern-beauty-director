# Cosplay展会摄影

## 定位

用于生成具有国际商业摄影质感的 Cosplay 展会摄影系列。

核心目标：

- 官方游戏角色宣传摄影
- AAA 游戏角色 Cosplay
- Vogue / Harper's Bazaar 时尚大片
- ChinaJoy / Bilibili World / Tokyo Game Show 官方展会摄影

不是：

- 传统漫展随拍
- 廉价 Cos 摆拍
- 网红棚拍
- 页游宣传图
- 拼图海报
- 低质淘宝 Cos 服展示

人物仍然属于东方美人系列：年轻成年东亚女性、真实摄影感、人物第一视觉中心。

## 固定摄影系统

- 比例：9:16 竖构图。
- 风格：超写实摄影、RAW Photo、官方展会摄影、Vogue Editorial、Luxury Fashion Campaign。
- 镜头：85mm portrait lens，f/1.8，浅景深，轻微长焦压缩。
- 色调：电影级色彩，Kodak Portra 气质，干净低噪点。
- 景别：默认大腿以上中景；人物占画面约 70%。
- 机位：眼平或轻微低机位，摄影师近距离拍摄。
- 禁止：超广角、怼脸、鱼眼、拼图、人物过小、背景抢戏。

## 展馆系统

固定环境为大型动漫游戏展馆。

可用场景：

- ChinaJoy 风格官方游戏展台
- Bilibili World 风格互动展区
- Tokyo Game Show 风格发布展台
- 游戏角色摄影区
- LED 主舞台侧区
- 官方主题布景区
- 游戏试玩区外侧
- 品牌发布墙前

背景元素：

- 官方游戏展台
- 游戏海报
- Booth Logo 弱化软焦
- 少量摄影师
- 少量观众
- LED 灯光
- 展馆顶部灯光

背景规则：

- 轻微虚化。
- 展馆存在感服务人物。
- 不要大量人群。
- 不要广告文字抢戏。
- 不要杂乱展板。

## 人物系统

年龄：20-26 岁成年女性。

默认：东亚女性。

面部：

- 小鹅蛋脸或柔和鹅蛋脸。
- 自然骨相。
- 五官真实。
- 大眼但不过度动漫化。
- 真人摄影感。
- 不网红脸。
- 不 AI 塑料脸。

肤质：

- 健康白皙。
- 真实皮肤纹理。
- 适度自然高光。
- 不过度磨皮。

身材：

- 这是硬锁定，不被展会、角色职业、正式服装或安全改写覆盖。
- 健康丰润。
- 富有生命力。
- 腰臀自然过渡。
- 肩颈舒展。
- 锁骨柔和。
- 胸部自然饱满。
- 比例协调。
- 腿部修长。
- 服装必须通过剪裁保留女性化曲线。
- 正式覆身不等于宽松遮挡身材。
- 即使是长外套、战术服、长裤或高领角色服，也要保留自然胸线、收腰结构、腰臀过渡和整体圆润体积。

禁止：

- 病态骨感。
- 纸片人。
- 平板身材。
- 过度清瘦。
- 宽松服装完全吞掉身体线条。
- 肥胖化。
- 极端沙漏。
- 夸张胸腰比。
- 幼态化。
- 现实职业制服恋物化。

## 多张生成硬规则

当用户要求“两张 / 一组 / 多张 / 随机”时：

1. 不得连续使用同一个风格簇。
2. 不得连续使用同一种展馆子空间。
3. 不得连续使用同一种服装轮廓。
4. 不得连续使用同一种姿态。
5. 不得连续使用同一种主道具。
6. 至少一张必须不是标准站姿看镜头。
7. 如果生成两张，优先使用强对比组合：
   - 科技未来系 + 赛事商业系。
   - 官方战斗系 + 悬疑叙事系。
   - 舞台偶像系 + 科技未来系。
   - 东方幻想系 + 赛事商业系。
8. 两张图不应该只是“同一个展台、同一种角色服、同一个中景、换个颜色”。

## 服装系统

核心原则：

- 少即是多。
- 高级剪裁。
- 明确轮廓。
- 材质统一。
- 现代游戏角色宣传图品质。

推荐材质：

- 丝缎。
- 哑光皮革。
- 精细金属件。
- 少量水晶点缀。
- 少量披肩。
- 少量腰封。
- 高级弹力面料。
- 细腻装甲片。

轮廓建议：

- 修身外套 + 高腰短裙，保留自然腰臀线。
- 哑光皮革连体战术套装，剪裁贴合但不色情化。
- 丝缎披肩 + 结构化胸甲，胸线自然饱满但不夸张。
- 短款角色夹克 + 长靴。
- 礼服式游戏角色战袍。
- 轻装女骑士剪裁。
- 未来偶像舞台装。
- 展会官方宣传角色服。

禁止：

- 廉价 Lolita。
- 大量蕾丝。
- 十几层裙摆。
- 大量欧根纱。
- 塑料感面料。
- 廉价淘宝 Cos 服。
- 透明暴露。
- 情趣化。
- 过度开孔设计。
- 过度宽松导致纸片人。
- 为了保守而取消女性曲线。

## 配色系统

每次固定一种主配色，不超过三种主体色。

推荐：

- 蓝白银：清冷梦幻。
- 红黑金：高贵华丽。
- 紫银：未来幻想。
- 黑金：成熟高级。
- 白金：圣洁典雅。
- 青绿金：东方幻想。
- 酒红金：贵族气质。
- 黑白红：官方赛事感。
- 银灰蓝：高科技冷感。

规则：

- 主色清晰。
- 金属和水晶只做点缀。
- 不要荧光色污染脸部。
- LED 背景不得压过人物肤色。

## 姿态系统

避免所有图片都是站着、看镜头、微笑。

站姿：

- 回头。
- 微侧身。
- 单腿重心。
- 整理发丝。
- 举角色道具。
- 扶腰封或披肩。

坐姿：

- 坐展台。
- 坐摄影区布景。
- 坐高脚椅。
- 坐官方主题布景。
- 半靠展示台。

互动：

- 向摄影师挥手。
- 看向同伴。
- 与观众互动。
- 整理披肩。
- 展示角色武器。
- 接过工作人员递来的角色卡。
- 低头检查道具灯效。

多人：

- 两人同行。
- 三角构图。
- 相视而笑。
- 并肩行走。
- 官方合影。

多人原则：

- 两人最佳。
- 三人可用。
- 超过三人慎用。
- 每增加一个人物，摄影师后退一步。
- 保持自然距离。
- 不要贴脸。
- 不要全部看镜头。
- 人物必须不同脸、不同发型、不同姿态。

## 构图系统

推荐：

- 人物约 70%。
- 中景。
- 大腿以上。
- 85mm。
- 眼平视角。
- 轻微长焦压缩。
- 背景自然虚化。
- 人物永远第一视觉中心。
- 有呼吸感。

禁止：

- 怼脸。
- 拼图。
- 多人挤在一起。
- 人物太小。
- 展台文字抢主体。
- 大量观众遮挡。

## 光线系统

展馆：

- LED 主光。
- 展馆顶部柔光。
- 官方展台屏幕光。

人物：

- 暖色主光。
- 柔和补光。
- 自然肤色。

边缘：

- 柔和轮廓光。
- 不过曝。
- 不塑料。

整体：

- 电影级。
- 高级商业摄影。
- 阴影自然。
- 干净低噪。

## 导演检查

生成前检查：

- 是否仍然是年轻成年东亚女性东方美人。
- 是否是官方展会摄影，而不是普通漫展随拍。
- 是否有明确原创角色方向。
- 服装是否高级、材质统一、轮廓清晰。
- 背景是否轻微虚化且不杂乱。
- 人物是否占画面约 70%。
- 是否避免所有人看镜头。
- 道具语义是否清晰。
- 手、脚、腿、道具是否有负面约束。
