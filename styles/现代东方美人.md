# 现代东方美人

Use this reference when the user asks for 写实东方美人, 现代东方审美, 新中式, 东方高级感, magazine portrait, cinematic portrait, fashion editorial, commercial beauty image, or non-fantasy Eastern beauty. If the user asks for SweetHomeGirl, 甜系纯欲生活写真, or a girlfriend-style lifestyle character, use `styles/甜系纯欲生活写真.md` first and treat this file as optional supporting style guidance.

## Core Rule

Do not force mythology, hanfu, magical artifacts, or wonder-scale environments. Build the image through face temperament, styling, garment silhouette, fabric, posture, light, lens, background, and restraint.

Use `styles/东方美人审美系统.md` when the user asks for a specific beauty temperament such as 柔美, 妩媚, 成熟, 优雅, 高贵, 清冷, 温婉, 英气, 神性, or 慵懒.

A strong modern Eastern beauty prompt should include:

- Adult subject boundary.
- Face temperament and facial structure.
- Styling route: new Chinese style, fashion editorial, cinematic portrait, beauty campaign, quiet luxury, or cultural-modern fusion.
- Garment silhouette and fabric behavior.
- Posture, gaze, and one meaningful hand action.
- Motivated light and lens language.
- Background with taste and spatial depth.
- Controlled palette.
- Clear exclusions for over-retouching, cheap costume cues, and sexualized framing.

## Subject Notes

- Face temperament: 清冷, 温婉, 英气, 神性克制, 华贵, 疏离, 知性, 松弛.
- Facial structure: oval classical face, lean high-fashion face, calm almond eyes, phoenix eyes, soft natural brows, restrained lips, natural skin texture.
- Makeup: clean base makeup, subtle rouge, thin eyeliner, muted red lip, pearl highlight, no heavy cosplay makeup.
- Hair: low bun, clean side part, loose black hair, wet-look editorial hair, jade hairpin as a small accent.
- Posture: straight neck, relaxed shoulders, seated profile, half turn, one hand touching collar, holding cup, adjusting earring, touching rain glass, resting on carved wood.

Avoid treating "beauty" as body exposure. Use expression, posture, tailoring, fabric weight, and light to create attraction.

## Wardrobe and Materials

- New Chinese style: cross-collar blouse, qipao-inspired dress, silk jacket, long coat, linen suit, wide-leg trousers, jade button, frog closure.
- Classical accents: standing collar, cloud shoulder reference, ink-wash embroidery, bamboo pattern, pearl earring, jade ring.
- Fabrics: silk, satin, linen, wool, organza overlay over opaque layers, matte crepe, lacquered leather accent.
- Avoid: cheap costume hanfu, transparent revealing fabric, lingerie styling, random festival costume, plastic-looking satin.
